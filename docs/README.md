# EXACT 2026 Docs

This folder summarizes two things:

1. What the EXACT 2026 competition currently requires.
2. How this repository is built to satisfy those requirements.

Documents:

- `competition-overview.md`: concise summary of the official competition page.
- `project-architecture.md`: repo-focused explanation of the API, LangGraph flow, deployment, and tests.
- `competition-fit.md`: mapping between competition requirements and the current implementation, including known risks.

Recommended reading order:

1. `competition-overview.md`
2. `project-architecture.md`
3. `competition-fit.md`
